# SVG Path Animation in Next.js using GSAP

![image](https://github.com/aashishpanthi/svg-path-animation/assets/60884239/bde7d1a0-7feb-4162-a64f-a3bb7704e2fb)

## Introduction

This project demonstrates how to create SVG path animations using GSAP (GreenSock Animation Platform) in a Next.js application. You will learn how to animate SVG elements along a path on scroll events.

## Demo

You can see the live demo [here](https://gsap-svg-path-animation.netlify.app).

## Tech Stack

- **Next.js** - A React framework for server-side rendering and static site generation.
- **GSAP (GreenSock Animation Platform)** - A powerful JavaScript library for creating animations.

## Getting Started

### Clone the Repository

To get started, clone the repository to your local machine:

```bash
git clone https://github.com/aashishpanthi/svg-path-animation.git
```

## Install Dependencies
Navigate to the project directory and install the required dependencies:

```bash
cd svg-path-animation
npm install
```

## Start the Development Server
Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
